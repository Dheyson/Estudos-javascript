# Flexbox

## "Flex-items"

### Flex Grow

Mostra a possibilidade de um item crescer.

{% embed url="https://codepen.io/dheyson\_alves/pen/abvKVJe?editors=1100" %}

No CodePen acima, mostro como utilizar o flex-grow. 

* **flex-grow: 0** respeita a largura do conteúdo.
* **flex-grow: 1** irá ocupar 100% do container. Dependendo do tamanho dos outros items. Respeitando as largura de todos.
* **flex-grow: 2** tentará ocupar o dobro do espaço do menor tamanho. Como no exemplo acima.

**OBS: justify-content não funciona em items com flex-grow definido.**
